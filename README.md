# arduino-PS2-holonomic-code
In this repo, there is code of ps2 controlled arduino based holonomic drivetrain 

Hello guys,

 Pins of motors are :

Motor 1:
            Direction pin :13
            Pwm Pin:12
Motor 2:
            Direction pin :11
            Pwm Pin:10
Motor 3:
            Direction pin :9
            Pwm Pin:8
Motor 4:
            Direction pin :7
            Pwm Pin:6

I am using md10c motor driver and Dc geared motor.
